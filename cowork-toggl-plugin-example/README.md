# Introduction 
This repository contains an example Microsoft 365 Copilot Cowork Plugin which will allow users to use a custom Toggl MCP Server to fill out their timesheets and tasks in Toggl from a Microsoft 365 Calendar.

# Skills Development
These skills are built as plugins following the guidance from here
https://learn.microsoft.com/en-us/microsoft-365/copilot/cowork/cowork-plugin-development#build-a-plugin-from-scratch
